Ce projet contient un jeu de type TowerDefense dans lequel le joueur peut placer des défenses via le clique gauche de la souris.
Il peut également déplacer un personnage et le faire attaquer les différents ennemie via les touches zqsd et la barre d'espace.
Le but de ce projet était de créer les bases d'un TowerDefense en partant de rien, des améliorations pourrait être apportées.

Pour jouer au jeu, télécharger le dossier "executable" et lancer la commande java -jar TowezDefense.jar
Vous avez la possibilité de lancer le jeu en mode developpeur avec la commande java-jar TowerDefense.jar root
